# Chronos Develop Plan

The authoritative, living develop plan is kept as a manuscript under version
control:

- [`docs/plans/manuscripts/20260619-chronos-develop-plan.md`](plans/manuscripts/20260619-chronos-develop-plan.md)

It covers project positioning, the target use case (restricted data centers that
allow only HTTPS egress), the server transport modes, the v1 scope and
non-goals, the repository layout, component design, the chrony SOCK refclock
wire format, the Docker packaging decision (a single combined image), and the
development milestones (M1–M12).

This file is intentionally a pointer so the plan is maintained in exactly one
place.
